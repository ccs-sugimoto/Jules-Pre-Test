# テトリスゲーム

シンプルなテトリスゲームです。Webブラウザで動作します。

## 利用方法

1.  このリポジトリのファイル一式（`index.html`, `script.js`, `style.css`）を同じディレクトリにダウンロードします。
2.  `index.html` ファイルをWebブラウザで開きます。
    - 例: Chrome, Firefox, Safari, Edgeなど

## 操作説明

-   **← (左矢印キー)**: テトリミノを左に移動します。
-   **→ (右矢印キー)**: テトリミノを右に移動します。
-   **↑ (上矢印キー)**: テトリミノを時計回りに90度回転します。
-   **↓ (下矢印キー)**: テトリミノを下に速く落下させます（ソフトドロップ）。
-   **Enterキー**: ゲームオーバー時に、ゲームをリスタートします。

## 開発者向け情報

-   **ログ**: 主要なゲームイベントはブラウザの開発者コンソールのログに出力されます。
    - ゲームの開始、テトリミノの生成・固定、キー入力、ライン消去、スコア計算、ゲームオーバーなど。
