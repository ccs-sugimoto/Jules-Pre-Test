# テトリスゲーム

シンプルなテトリスゲームです。Webブラウザで動作します。

## 利用方法

1.  このリポジトリのファイル一式（`index.html`, `script.js`, `style.css`）を同じディレクトリにダウンロードします。
2.  `index.html` ファイルをWebブラウザで開きます。
    - 例: Chrome, Firefox, Safari, Edgeなど

## ゲームの流れ

1.  ゲームを起動すると、まずタイトル画面が表示されます。
2.  タイトル画面にある「ゲーム開始」ボタンをクリックすると、ゲーム画面に遷移し、テトリスが開始されます。
3.  ゲームオーバーになると、自動的にタイトル画面に戻ります。

## 操作説明

**タイトル画面では、マウスで「ゲーム開始」ボタンをクリックしてゲームを開始してください。キーボード操作では開始できません。**

**ゲームプレイ中:**

-   **← (左矢印キー)**: テトリミノを左に移動します。
-   **→ (右矢印キー)**: テトリミノを右に移動します。
-   **↑ (上矢印キー)**: テトリミノを時計回りに90度回転します。
-   **↓ (下矢印キー)**: テトリミノを下に速く落下させます（ソフトドロップ）。
-   **スペースキー**: テトリミノを一気に下に落下させます（ハードドロップ）。

**ゲームオーバー後:**

-   ゲームオーバーになると、自動的にタイトル画面に戻ります。
-   タイトル画面が表示されている状態で **Enterキー** を押すと、タイトル画面に戻ります（特にゲームがリスタートするわけではありません。ゲーム開始は常にボタン操作が必要です）。

## 開発者向け情報

-   **ログ**: 主要なゲームイベントはブラウザの開発者コンソールのログに出力されます。
    - ゲームの開始、テトリミノの生成・固定、キー入力、ライン消去、スコア計算、ゲームオーバーなど。

## 現在のゲームの主な挙動

-   ゲーム開始時にはタイトル画面が表示されます。
-   スタートボタンを押すことでゲームが開始されます。
-   ゲームオーバーになるとタイトル画面に戻ります。
-   スコア、経過時間が表示されます。
-   次に降ってくるテトリミノ（ネクストミノ）が表示されます。
